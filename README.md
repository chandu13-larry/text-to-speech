# text-to-speech-from-python
🎧 Text to Speech from PDF (Python)

A simple Python desktop application that extracts text from a PDF file and converts it into speech using Google Text-to-Speech. Built with Tkinter for GUI.



🚀 Features
📄 Select any PDF file
🔊 Convert PDF text to speech
🎧 Save output as MP3
🖥️ Simple Tkinter GUI
⚡ Easy one-click playback

🛠️ Technologies Used
Python 🐍
Tkinter (GUI)
gTTS
Apache Tika
OS module


📦 Installation

Clone the repository:

git clone https://github.com/your-username/text-to-speech.git
cd text-to-speech

Install dependencies:

pip install gTTS tika

Make sure Java is installed:

java -version
